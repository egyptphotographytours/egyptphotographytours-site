埃及烹饪:5 000年的烹饪旅程历经历史

埃及烹饪是世界上最古老、最多样化、文化上最有意义的烹饪传统之一,有记载的食谱和烹饪技巧可追溯到法老时代的五千年。 这种活的烹饪传统吸收了古老文明的影响,包括法老、希腊、罗马、科普特、阿拉伯、奥托曼和现代地中海文化,创造了独特的丰富和有味的胃学景观,这种景观在继续演变的同时保持其深厚的历史根源。 与许多世界烹饪不同,埃及食品提供了一种不折不扣的烹饪品种,允许现代餐饮品品尝克列奥帕特拉、拉姆谢斯二世和金字塔建筑商熟悉的味道。

在埃及摄影旅行中,我们认为食品摄影是记录埃及文化的最亲密和真实的方式之一。 我们以烹饪为主的摄影旅行提供了独家进入传统厨房、街头食品销售商、香料市场和捕捉埃及烹饪灵魂的家庭餐饮体验。 从开罗的Khan el-Khalili市场繁忙的食品摊位到俯瞰尼罗河的尖端餐馆,我们以摄影专业知识和烹饪知识指导你穿过埃及的胃道景观。

埃及菜食的历史演变:从法老到现代餐桌

了解埃及食物需要欣赏其历史层面。 古埃及烹饪以面包和啤酒为基础(由大麦和小麦制成 ) , 补充了来自尼罗河的蔬菜、水果、豆类和鱼类。 希罗格两栖证据表明了先进的烘烤技术和详细的食谱。 希腊和罗马时期引入了橄榄油、葡萄酒和新的烹饪方法。 光学基督教带来了禁食传统,创造了丰富的素食品种。 伊斯兰征服带来了香料、大米和新的肉类制作技术。 奥托曼规则增加了甜食和糕点。 现代埃及烹饪在保持其独特性的同时结合了全球影响。

埃及早餐订书店多数是蔬菜

埃及早餐是用于维持体力劳动的大量富蛋白质食品。 与西方早餐不同,埃及早间饭是美味的、有心的,而且往往是在传统粘土锅中夜间烹制的豆类。 早餐店在黎明前开张,到明天中午仍然很忙。

ful medames 纤维 medames必须尝试 蔬菜

无可争议的埃及民族菜盘,富饶的medamames由在巨大的铜锅中浸泡8-12小时的慢熟的法瓦豆组成。 里面有橄榄油、柠檬汁、大蒜、蛋蛋,还有可选的塔希尼、硬煮蛋或新鲜蔬菜。 历史记录显示金字塔工人大量消费。 每个地区都有不同之处:亚历山大加了番茄,卢克索加了更多的大蒜,阿斯旺加了日期糖。

Taameya(埃及法拉费)必须尝试 维卫星Name

埃及对中东烹饪的独特贡献与Levantine falafel不同,它使用fava豆而不是小鸡豆。 其结果是更轻、更脆、更香味的肉饼,通常配有帕斯利、cilantro、leek和香料。 其服务于配有沙拉和Tahini酱的Pita面包。 最有经验的塔米亚专卖店,他们每天早上都点菜。

Feteer Meshaltet(埃及层糕点)

古老的糕点可追溯到法老时代,即fateer,由数百个纸质面团层组成,用澄清的黄油(samma ) 分离。 简洁的版本凸显了技术技能,而现代的变异包括奶酪、薄荷肉或蜂蜜和坚果等甜料。 专门化的食品店24小时营业,早餐时活动高峰。

Gibna Domiati(多米亚提奶酪)

埃及用水牛或奶牛制成的白色乳酪(含盐水桶的锡罐中已老化 ) 。 埃及在早餐时用面包、西红柿和薄荷做饭。 三角洲地区生产最佳版本,确定口味的强度从轻度(3个月)逐渐老化到尖锐(2年以上 ) 。

主要课程和家庭膳食

埃及的主菜展示了烹饪的复杂性,包括慢熟的炖菜、垫菜和精美的饭菜,它们构成了家庭餐和庆祝活动的核心。 烹饪技术强调通过扩大蒸馏和谨慎的香料平衡来分层调味。

科沙里国民菜 维卫星Name

埃及最终的舒适食品和烹饪象征着埃及在碗中的历史。 大量的米饭、棕扁豆、鸡尾豆和ditalini意大利面粉都充斥着辣番茄酱、大蒜醋和脆煎洋葱。 每个成分都是分开煮的,然后加在一起。 Koshari商店从街摊到专为数千人服务的餐厅,从街道摊子到专为数千人服务的餐厅不等。 我们开罗的食品旅游包括访问具有摄影机会的具有历史意义的科沙里企业。

Molokhia(朱特Leaf Soup)

这种古老的青汤由精细切碎黄麻叶制成,具有类似于奥克拉的独特的粘土纹理。 通常使用兔子或鸡或米和米,摩洛西亚和法老的时段都配有被认为是皇家菜。 准备方式各有不同:开罗版是汤味的,亚历山大加了番茄,上埃及让它更干燥。叶子总是用特殊的双手刀(摩克拉塔式刀 ) 以光源的有节奏动作切碎。

Mahshi(已装填的蔬菜)蔬菜选项

一种盛宴,配有蔬菜(succhini、茄子、胡椒、卷心菜、葡萄叶),配有草床米饭混合物。 肉品包括小羊肉。 每个家庭都有代代相传的食谱。 盛装蔬菜在锅里装满了肉汤和缓慢烹调。 展示多彩蔬菜是食品摄影的理想选择。

法塔美食节

用于庆祝(爱德、婚礼、出生)、脂肪层大米、油炸面包和肉类(通常为羊肉)的预留品包括大蒜醋和番茄酱。 对比鲜明的纹理和口味代表着繁荣。 地区差异包括亚历山大的海鲜脂肪和卢克索的辣味版本。

赛亚迪雅(以大米为食)

埃及沿海的特色菜菜有白鱼(通常是海贝斯或鱼),然后用焦糖洋葱和大米烤成番茄酱。 亚历山大和赛义德港提供了最佳版本。 黄稻的金褐鱼造就了惊人的摄影成份。

埃及甜食和甜甜文化

埃及甜蜜传统:从古代蜂蜜到现代糖果

埃及甜点代表着世界上最古老的甜点传统之一,在法老时代有蜂蜜甜点记录。 奥托曼帝国引入了糖浆浸泡的糕点,而地中海则增加了奶制品甜点。 埃及甜点平衡了极甜与芳香香调味品和花纹对比。

签署 埃及甜品

乌姆阿里必须尝试

埃及对面包布丁的回答是以13世纪苏丹命名的。 层层的糕点、牛奶、坚果和葡萄干烤到金色。 服了温暖的、常常配奶油的。 在蒸汽上升的烤箱中,最新鲜的被拍到。

塞莫利娜蛋糕浸在玫瑰或橙花糖浆中,往往堆满杏仁或椰子。 质素从稠密和潮湿到光和碎屑不等。 适合糖浆吸收的宏观摄影。

罗兹·贝尔·拉班

埃及饮料:从传统到当代

传统埃及饮料

埃及茶( Shai)国家饮料

茶叶文化渗透到埃及社会,茶叶店成为社会枢纽。 筹备情况各有不同: "shai na'na" (用薄荷)、 "shai libton" (利普顿品牌)、 "shai koshary" (非常强烈 ) 。 拍摄传统杯子、 samovars 以及社会茶饮场景。

土耳其咖啡(阿瓦)

咖啡馆(阿哈瓦)是男性主导的社会空间。 摄影机会包括传统的黄铜锅、泡沫涂料和咖啡财富阅读。

茶(卡尔卡德)维卫星Name

鲜红茶由干枯的木豆花制成,热的或冷的。 天然的无咖啡因和丰富的维生素C。 深红的红色给传统玻璃软件制造了惊人的摄影成份。

萨哈拉布

用兰花根粉、牛奶和肉桂、坚果和椰子制成的温和、奶油浓润的冬季饮料。 冬季几个月的街头销售商提供了光源化准备,冷空气中蒸汽不断上升。

果汁糖(Aseer Asab)维卫星Name

新鲜的甘蔗汁,是街道摊位上可买到的甜甜的、充满活力的饮料。 紧迫的机器和新鲜的甘蔗可以产生动态的动作镜头。

旅行者食品安全和健康考虑

埃及基本食品安全准则

虽然埃及食品一般是安全的,但遵循这些准则可以确保健康烹饪经验。 我们的旅游仔细选择了在真实性和卫生标准之间取得平衡的机构。

1. 水安全

只饮瓶装水,避免自来水,包括饮料中的冰。用瓶装水刷牙。要谨慎使用新鲜的果汁,确保用瓶装水制成。

2. 街头食品选择

选择繁忙的更替率高的摊位。 完全熟熟的食品的选项服务非常热。 避免生蔬菜, 除非您可以自己剥开这些蔬菜。 谨慎对待街头销售商的奶制品 。

3. 餐厅的挑选

与其他顾客一起寻找清洁设施; 检查洗手间清洁程度,作为厨房标准的指标; 比较容易看到食物准备的地方。

4. 个人卫生

经常洗手或使用洗手剂; 在用餐前携带湿巾; 在吃街头食物时避免碰面; 尽可能使用用具。
