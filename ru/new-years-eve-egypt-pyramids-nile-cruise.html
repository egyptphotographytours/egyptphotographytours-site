Новогодняя фотография пирамид – профессиональные секреты

Как специалисты по фотографии, мы знаем об уникальных трудностях съемки фейерверков над древними памятниками в условиях большого скопления людей:

Основы оборудования

  • Штатив:Не подлежит обсуждению для резких выстрелов фейерверков. Легкое углеродное волокно рекомендуется для путешествий.
  • Объектив:Широкоугольный (16–35 мм) для контекстной съемки; телеобъектив (70-200 мм) для деталей пирамид во время фейерверков.
  • Настройки:Ручной режим, ISO 400, f/8, ручной режим с дистанционным триггером (выдержка 2–4 секунды).
  • Резервное копирование:Дополнительные батарейки (на холоде разряжается), карты памяти, ткань для линз (пыль пустыни).

Позиционная стратегия

Классический выстрел — фейерверк прямо над вершиной пирамиды — требуетк юго-западу от Сфинкса. Наши частные туры закрепляют эти позиции за несколько месяцев вперед. Посетителям следует прибыть к 19:00 и занять возвышенность (небольшие холмы возле Сфинкса дают небольшое преимущество по высоте).
